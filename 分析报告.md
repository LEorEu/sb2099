## 3. 技术栈推断

- **前端**：单页应用，纯 Hash 路由（`#/home`、`#/aichat`...），是 Vue/React SPA 的典型表现；页面壳由前端渲染，后端只提供 JSON。
- **CDN / 防护**：使用 Cloudflare（请求中可见 `cdn-cgi/challenge-platform`、`cdn-cgi/rum`）。
- **统计**：接入 Microsoft Clarity（`e.clarity.ms/collect`）和 `api.tongjiniao.com`（疑似第三方统计，本次返回 503）。
- **静态资源 CDN**：`cdn.hguofichp.cn`（油猴脚本下载）、`pic.imgdb.cn` / `pic1.imgdb.cn`（标签图标）、`oss.5eplay.com`（选手 / 战队头像，复用 5EPlay 的 CDN）。
- **后端 API 入口**：`https://hguofichp.cn:10086/machine/*` —— 非主域，10086 端口，与作者另一域名 `hguofichp.cn` 同源；接口风格统一为 `{code, msg, data}`，明显是 **Java/Spring Boot**（`code:200, msg:"请求成功"` 是国产 Spring Boot 脚手架（如若依 RuoYi）的默认响应壳，而且 `dictType: "machine_tags"` 也是 RuoYi 的字典表设计）。
- **后端语言初判**：高度可能是 **RuoYi 衍生的 Spring Boot + MyBatis** 系统；字段命名（`pageNum / pageSize / lastPage`、字典三件套 `dictCode/dictLabel/dictValue/dictType`）几乎是 RuoYi 原样照搬。

## 5. 亮点功能

1. **一键复制 / 油猴一键发送**
   每条烂梗整卡可点即复制；油猴脚本 `cdn.hguofichp.cn/.../斗鱼玩机器烂梗收集.user.js` 注入斗鱼直播间，把发送按钮直接接到这个语料库，是站点的核心入口。已支持斗鱼新 UI。
2. **AI 造梗**
   接入 **397B-A17B** 大模型（明显参数量描述指向 MoE 架构，类似 Qwen3-A22B 系列），每天 50 次限额，可选标签 + 查看「思考过程」，是 LLM + 站点语料结合的玩法。
3. **CS2 赛事数据榜**
   `布雷德 15 勇士`（对位 Donk 击杀 / 被杀） / `超级逮虾户战报`（NiKo 逮虾命中与反夹）是基于 HLTV / 5EPlay 数据二次加工的轻量数据可视化，把直播间梗和真实比赛数据联动。
4. **屏蔽词收集**
   收集斗鱼平台对该直播间生效的关键词屏蔽词，反映直播平台审查现状，社区共建 ↔ 平台规则的一种小型对抗记录。
5. **词云 + 年度 TOP20**
   词云接口 `/WordCloud` 提供 TOP100 频词，配合「年度 TOP20 烂梗」做季节性回顾，强化"年鉴"属性。
6. **关联赛事库**
   `InProgressMatch` 接口决定投稿框上的「关联赛事库」开关是否可用，赛季有比赛时投稿会自动归档到赛事库 → 这是一个有意识做"按事件归档"的设计，比纯时间流要好。

## 6. 运营 / 商业化观察

- **服务器倒计时**+**已运行天数** 显式展示在底部，营造"小作坊"的亲近感，同时暗示需要捐助。
- **金主要求注册 150 人** 按钮 + tooltip「甲方爸爸…凑凑人头吧」：明确告诉用户**广告金主有 KPI**，要求 150 个新注册。
- 推广位指向 `yousheng186.com`（带 `promotionCode=6657`），看上去是博彩/竞猜导流（对应屏蔽词里有「博彩」也合理——既被斗鱼屏蔽，也是站方主要变现）。这块是该站最大的合规风险点，浏览 / 二次开发时要避开。
- 友情链接：`dgq63136.cn`、`星空背景`，看起来是同主播圈的姐妹站。

## 7. 可借鉴的工程要点（若做 sb2099 同类型站）

- **后端**：用 RuoYi / Spring Boot 起步成本极低，字典表 + 分页 + 简单审核就能撑起 UGC 语料库。
- **协议设计**：弹幕表字段精简（`barrage / tags / cnt / submitTime`），用字典表存标签 → 易迁移、易统计。
- **CDN 与图床外挂**：标签头像直接放 `imgdb.cn`，省自己存储；选手 / 战队图标复用 `oss.5eplay.com`，能蹭就蹭。
- **入口杠杆**：站点本身 + 油猴脚本 + LLM API 三件套；其中油猴脚本是最关键的转化漏斗，把"看烂梗"变成"用烂梗"，反向给主播直播间引流，再形成新内容。
- **季度 / 年度榜单**：把 UGC 沉淀做成"年鉴 / 名人堂"是黏住老用户的便宜手段。
- **风险**：博彩 / 桃色广告强依赖、版权（直播弹幕收录 / 主播肖像）、平台屏蔽词反向公示 ——
  如果你只是做"自用语料库 / Demo"，把广告位和注册逼单全去掉就行；如果要长期对外开放，需考虑成人/赌博广告合规问题。
